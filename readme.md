# Decola Tech 2024

Este repositório reúne os desafios e exercícios realizados durante o programa Decola Tech 2024.

## Estrutura do Projeto

### Desafios

- **construindo-sistema-para-um-estacionamento**: Implementação de um sistema para controle de estacionamento, utilizando fundamentos da linguagem .NET.
- **criando-um-sistema-e-abstraindo-um-celular-em-poo**: Exercício de abstração e modelagem de um celular usando Programação Orientada a Objetos.
- **implementando-validacoes-de-testes-unitarios**: Desafio focado em validações e testes unitários, com exemplos práticos.

### Exercícios

- **dominando-a-linguagem**: Exercícios para aprofundar o domínio da linguagem .NET, incluindo manipulação de arquivos e tratamento de exceções.
- **introducao-da-linguagem**: Atividades introdutórias para familiarização com a sintaxe e conceitos básicos da linguagem.
- **orientacao-a-objetos**: Exercícios sobre conceitos de orientação a objetos, interfaces e herança.
- **testes-unitarios**: Prática de criação e execução de testes unitários para garantir a qualidade do código.
